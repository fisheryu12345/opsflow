📝 在 CLAUDE.md 添加

## OPSflow Style Rules:
- Use @use 'styles/opsflow-variables' for all SCSS
- Dialog: 加 class opsflow-dialog 自动获得统一样式
- Card: 用 .of-card class
- Animation: 用 .of-fade-in-up / .of-stagger-item
- 渐变: 用 $of-gradient-* 变量
- 新组件命名: kebab-case, 组件前缀 opsflow-

📁 新建页面模板

在 opsflow/styles/ 中会有一个
template-page.md 或 .vue 模板文件:

// 1. scss 导入
// 2. 对话框加 opsflow-dialog
// 3. 卡片用 .of-card
// 4. 列表用 stagger 动画
// 5. 空状态 image-size="40"

✅ 重构后的 AI 开发流程

  • AI 读 CLAUDE.md → 知道用 @use 'styles/opsflow-variables'
  • 写新组件 → 模板复制 + 变量引用 → 自动风格一致
  • 改旧组件 → 顺手替换成本地变量 → 逐次收敛
