条件编辑器 — 架构设计

数据流

// 用户操作流程
连接网关边 → PropertyPanel 显示简版条件
 → 点击"编辑条件" → ConditionDialog 弹窗
 → 选择变量 / 运算符 / 值 → 实时预览 ${...} 表达式
 → 点确定 → 写入 edge.data.condition

// 数据结构（不改变现有 pipeline_tree 格式）
edge.data.condition = "${node_1.cpu} > 80"
// 同时存储结构化条件供前端二次编辑
edge.data.conditionStruct = {
 logic: "AND",
 rules: [{
  source: "node_1",
  field: "cpu",
  op: ">",
  value: "80"
 }]
}

组件关系

PropertyPanel.vue (选中边时) ├── 标签下拉框（已有） ├── 条件只读显示 + [编辑条件] 按钮 │ ConditionDialog.vue ← 新组件 ├── ConditionRow.vue（单行：变量 + 运算符 + 值） ├── AND/OR 切换 + 添加行按钮 ├── VariablePicker.vue（变量选择下拉） ├── 表达式实时预览区域 └── 校验结果提示 useGraphCanvas.ts └── updateEdgeCondition(edgeId, struct) ← 新方法 解析 struct → condition 字符串 写入 edge.data // 后端无改动 — 条件字符串格式不变
