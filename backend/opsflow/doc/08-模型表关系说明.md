## 一、总览

| `template.py` | TemplateNode | `ops_template_node` | 模板节点（反范式化） |

## 二、模型详解

### 2.1 模板域（template.py）

#### FlowTemplate — `ops_flow_template`

编排模板主表，存储 AI 生成或人工创建的流程定义。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | AutoField | PK | 自增主键 |
| name | CharField(200) | NOT NULL | 模板名称 |
| pipeline_tree | JSONField | default=dict | Pipeline 定义（节点/边） |
| target_hosts | JSONField | default=list | 目标主机列表 |
| global_vars | JSONField | default=dict | 全局变量 |
| is_draft | BooleanField | default=True | 是否为草稿 |
| ai_original_tree | JSONField | default=dict | AI 原始生成树 |
| category | CharField(64) | blank=True | 分类 |
| tags | JSONField | default=list | 标签 |
| description | CharField(500) | blank=True | 描述 |
| hook_variables | JSONField | default=dict | 变量提升配置 |
| project | FK -> OpsProject | CASCADE, nullable | 所属项目 |
| created_by | FK -> User | PROTECT, nullable | 创建人 |
| created_at | DateTimeField | auto_now_add | 创建时间 |
| updated_at | DateTimeField | auto_now | 更新时间 |
| version | IntegerField | default=1 | 当前版本号 |
| snapshot | JSONField | nullable | 发布快照 |
| is_public | BooleanField | default=False | 是否公开 |
| project_scope | JSONField | default=list | 可见项目范围 |

**关键方法：**
- `publish_snapshot(user, version_note)` — 发布新版本，冻结 pipeline_tree/snapshot，创建 TemplateVersion 记录，提取子流程引用信息（target_template_id/version/name/variable_mapping/output_mapping）

## 三、模型关系图

```
OpsProject  ─┬── ProjectMember          (1:N 项目成员)
              ├── ProjectEnvironmentVariable (1:N 环境变量)
              ├── FlowTemplate           (1:N 模板)
              ├── FlowExecution          (1:N 执行)
              └── SchedulePlan           (1:N 调度计划)

FlowTemplate ─┬── TemplateVersion        (1:N 版本历史)
               ├── TemplateNode          (1:N 模板节点)
               ├── TemplateCollect       (1:N 用户收藏)
               ├── FlowExecution         (1:N 执行实例)
               ├── SchedulePlan          (1:N 调度计划)
               ├── ExecutionScheme       (1:N 执行方案)
               ├── WebhookConfig         (1:N Webhook)
               └── TemplateCategory      (N:1 分类)

FlowExecution ─┬── ExecutionNode         (1:N 执行节点)
                ├── NodeExecutionTrace   (1:N 节点轨迹)
                ├── AutoRetryStrategy    (1:N 重试策略)
                ├── NodeTimeoutConfig    (1:N 超时配置)
                ├── OpsLog               (1:N 审计日志)
                ├── WebhookLog           (1:N Webhook日志)
                ├── SchedulePlan         (N:1 调度计划)
                ├── FlowExecution        (N:1 父执行, self-FK)
                └── FlowTemplate         (N:1 关联模板)

SchedulePlan   ── FlowExecution          (1:N 执行历史)

WebhookConfig  ── WebhookLog             (1:N 投递日志)

TemplateNode   ── ExecutionNode          (1:N 执行节点引用)

User           ─┬── FlowTemplate         (1:N 创建人)
                 ├── FlowExecution       (1:N 执行人)
                 ├── TemplateCollect     (1:N 收藏)
                 ├── SchedulePlan        (1:N 创建人)
                 ├── WebhookConfig       (1:N 创建人)
                 ├── OperationRecord     (1:N 操作人)
                 ├── ApiToken            (1:N 创建人)
                 ├── ProjectMember       (1:N 项目成员)
                 └── OpsProject          (1:N 项目负责人)
```
