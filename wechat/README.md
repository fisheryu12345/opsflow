- **公众号适配：** 全部内联样式，已优化可直接粘贴到公众号编辑器
- **内容概要：**
  HVOB-MBI（High Volatility Opening Range Breakout - Market Breadth Index）是一个纯日内开盘区间突破系统，核心创新包括：
  - **三阶段工作流：** 夜盘突破（21:00-23:00，无MBI过滤）→ 跳空检查+MBI计算（9:00-9:30）→ 日盘突破（9:30-14:55，MBI过滤+时间衰减）
  - **MBI市场宽度指数：** 让观察池所有品种从三个维度（隔夜跳空、区间重心、突破方向）集体"投票"±1，映射到 [0,1] 确定多空权重
  - **品种评分筛选：** ATR%/振幅/量比/品种奖惩综合评分，每天动态选取得分前 5-10 个品种
  - **七层风控：** 品种筛选 → MBI权重 → 跳空拉黑 → 时间衰减 → 永久锚定止损 → 保本/移动止盈 → 14:55强制平仓
  - **风险敞口法仓位：** 固定 0.5% 单笔风险，波动越大开仓越少
  - 策略刚启动暂无实盘数据，文章重点放在三阶段工作流和MBI判断逻辑的科普讲解
