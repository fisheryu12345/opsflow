# 持仓管理功能说明

## 功能概述

持仓管理模块实时展示策略的持仓状态，包括合约代码、持仓方向、单位数、总手数、止损价、跟踪价格等关键字段。持仓数据由开盘任务自动维护，**只读展示，不支持手动修改**。

---

## 核心字段

| 字段 | 说明 | 来源 |
|------|------|------|
| `symbol` | 合约代码 | 开仓时写入 |
| `product_code` | 品种代码 | 开仓时写入 |
| `units` | 持仓单位数（0-3） | 开仓/加仓/平仓时更新 |
| `direction` | 持仓方向 | 1=多头，-1=空头，0=空仓 |
| `contract_total_position` | 此合约总持仓手数 | 开仓/加仓时累加 |
| `last_add_price` | 上次加仓价 | 加仓时更新 |
| `first_open_price` | 首次开仓价 | 开仓时写入 |
| `highest_close` | 持仓期最高收盘价（多头） | 每日收盘后更新 |
| `lowest_close` | 持仓期最低收盘价（空头） | 每日收盘后更新 |
| `stop_loss_price` | 当前动态止损价 | 每日收盘后计算更新 |
| `trend_info` | 趋势信息（ATR, factor, label） | 每日收盘后更新 |
| `latest_close_price` | 最新收盘价 | 每日收盘后更新 |
| `h20_price` / `l20_price` | 20日最高/最低价 | 每日收盘后更新 |
| `is_rollover_needed` | 是否需要移仓换月 | 合约同步时标记 |
| `protect_cost_enalbed` | 保本是否已启用 | 满足条件时开启（单向开关） |

---

## 筛选功能

- **查看持仓**：只显示 `contract_total_position > 0` 的记录
- **查看全部**：显示所有合约持仓记录（含空仓）
- **搜索**：支持合约代码、品种代码模糊搜索
- **方向筛选**：多头 / 空头 / 空仓
- **移仓筛选**：筛选 `is_rollover_needed=True` 的记录

---

## 数据流转

```
开仓成功 ──► 创建 PositionState（units=1, direction=±1）
加仓成功 ──► 更新 PositionState（units+=add_units, last_add_price=新价）
平仓成功 ──► 清空 PositionState（units=0, direction=0, 删除止损价等）
移仓成功 ──► 删除旧 PositionState，创建新 PositionState（新合约）

每日收盘 ──► 更新 highest_close / lowest_close / latest_close_price
         ──► 更新 stop_loss_price（动态跟踪止损 + 保本兜底）
         ──► 更新 trend_info / h20_price / l20_price
```
