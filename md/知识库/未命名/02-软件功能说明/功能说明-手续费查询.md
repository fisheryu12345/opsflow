# 手续费自动查询功能说明

## 功能概述

每日 14:58 自动查询所有活跃交易账户的当日累计手续费，同步更新 `TradingAccount.total_commission` 字段，使系统各模块可直接获取累计手续费数据而无需聚合查询。

---

## 定时任务

### 任务配置

| 项目 | 值 |
|------|-----|
| 调度时间 | 每日 14:58（工作日） |
| 任务 ID | `job_daily_commission_query` |
| 锁 Key | `lock:daily_commission`（Redis 分布式锁） |
| 源文件 | `backend/stock/scheduler/tasks_daily_commission.py` |

### 执行流程

```
job_daily_commission_query()
  │
  ├─ 1. 获取 Redis 锁 (lock:daily_commission)
  ├─ 2. 查询活跃账户列表 (TradingAccount.objects.filter(is_active=True))
  ├─ 3. 用第一个账户检查是否为交易日 (skip_if_not_trade_day)
  │
  └─ 4. 遍历每个活跃账户
        ├─ create_tqapi(account)  → 创建对应模式连接（模拟/实盘）
        ├─ api.get_account()      → 获取账户数据
        │     ├─ commission       → 当日累计手续费
        │     ├─ balance          → 当前权益
        │     ├─ available        → 可用资金
        │     ├─ float_profit     → 浮动盈亏
        │     ├─ margin           → 保证金
        │     ├─ risk_ratio       → 风险度
        │     └─ close_profit     → 当日平仓盈亏
        ├─ update_or_create DailyEquitySnapshot
        │     └─ 写入当日快照（含 commission 等字段）
        ├─ update_trading_account_total_commission()
        │     └─ 聚合所有 DailyEquitySnapshot.commission → 更新 TradingAccount.total_commission
        └─ safe_close_api(api)
```

### 好处

1. **提前捕获**：14:58 早于收盘计算（15:02），收盘任务异常时手续费数据仍保留
2. **去中心化访问**：各模块直接读 `account.total_commission`，无需跨表聚合
3. **双重保障**：收盘任务结束后也会同步调用 `update_trading_account_total_commission()` 确保最终数据准确

---

## 数据模型变更

### TradingAccount 新增字段

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `total_commission` | DecimalField(15,2) | 0 | 历史上所有交易的累计手续费总额 |

### 数据流

```
TqSDK api.get_account().commission (当日累计)
  → DailyEquitySnapshot.commission (日快照)
  → AccountPerformanceSummary.commission_total (绩效总览)
  → TradingAccount.total_commission (账户级快捷字段)
```

---

## 同步机制

### 收盘同步（tasks_daily_close.py）

收盘任务完成三层绩效更新后，调用：

```python
update_trading_account_total_commission(account)
```

确保 15:02 收盘计算后的最终手续费数据同步到 `TradingAccount.total_commission`。

### 绩效计算同步（performance.py）

`update_account_summary()` 计算 `commission_total` 后同步写入：

```python
account.total_commission = total_commission
account.save(update_fields=['total_commission', 'updated_at'])
```
