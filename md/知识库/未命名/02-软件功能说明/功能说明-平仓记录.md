# 平仓记录功能说明

## 功能概述

平仓记录模块记录每一笔平仓交易的完整信息，包括合约、方向、成交量、成交价、成本价、盈亏金额、持仓天数等。是绩效统计（胜率、盈亏比）和品种分析的数据源。

---

## 页面结构

```
┌─────────────────────────────────────────────────────────────┐
│  搜索栏：交易日期 / 合约代码 / 品种代码 / 平仓方向 / 盈亏金额   │
├─────────────────────────────────────────────────────────────┤
│  表格：平仓记录列表                                           │
│  ┌────────┬────────┬────────┬────────┬────────┬────────┐   │
│  │所属账户│合约代码│品种代码│平仓方向│平仓手数│平仓价  │   │
│  ├────────┼────────┼────────┼────────┼────────┼────────┤   │
│  │默认账户│rb2610  │rb      │多头平仓│8       │3250    │   │
│  └────────┴────────┴────────┴────────┴────────┴────────┘   │
│  继续：成本价 / 平仓盈亏（红绿色高亮）/ 交易日期 / 执行时间    │
│       持仓天数                                                │
└─────────────────────────────────────────────────────────────┘
```

---

## 核心字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `symbol` | 平仓时的合约代码 | rb2610 |
| `product_code` | 品种代码 | rb |
| `direction` | 平仓方向 | 1=多头平仓，-1=空头平仓 |
| `volume` | 平仓手数 | 8 |
| `exit_price` | 平仓成交价 | 3250.00 |
| `cost_price` | 持仓成本价 | 3200.00 |
| `pnl` | 平仓盈亏（已扣除手续费） | +4000.00（绿色）/ -2000.00（红色）|
| `trade_date` | 交易日期 | 2026-05-08 |
| `executed_at` | 执行时间戳 | 2026-05-08 09:02:15 |
| `holding_days` | 持仓天数 | 5.0 |

---

## 盈亏显示

- **盈利**：绿色高亮，前缀 `+`
- **亏损**：红色高亮，前缀 `-`
- 格式化保留 2 位小数

---

## 数据来源

平仓记录由以下场景自动创建：
1. **止损平仓**：`tasks_exit_before_close.py` 或 `tasks_daily_open.py`
2. **移仓换月**：`execute_rollover_order()` 平仓旧合约阶段

> 不支持手动添加或编辑，确保数据真实可信。

---

## 统计用途

平仓记录是以下指标的数据源：
- **品种胜率**：`GET /api/stock/symbol-win-rate/`
- **累计盈亏**：`GET /api/stock/cumulative-stats/`
- **三层绩效**：`core/performance.py` 中的胜率、盈亏比、连续盈亏等
