# 合约管理功能说明

## 功能概述

合约管理模块展示从 TqSdk 同步的全市场主力合约列表及元数据，提供查询和查看功能。
合约的交易激活已迁移到"交易品种管理"页面（基于 AccountContractConfig），此处仅作全局合约主表。

## 核心字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `exchange` | 交易所代码 | SHFE（上期所）、DCE（大商所）、CZCE（郑商所）、CFFEX（中金所）、GFEX（广期所） |
| `product_code` | 品种代码（不带年份） | rb、MA、IF |
| `symbol` | 当前主力合约代码 | SHFE.rb2610 |
| `name` | 合约中文名称 | 螺纹钢2605 |
| `volume_multiple` | 合约乘数（吨/手或点/手） | 螺纹钢=10，股指IF=300 |
| `price_tick` | 最小变动价位 | 螺纹钢=1，黄金=0.02 |
| `min_position` | 交易所限制最小开仓手数 | 通常=1，部分品种可能有特殊限制 |
| `night_trading` | 是否有夜盘交易 | true/false |

> 合约的交易激活控制已迁移到 `AccountContractConfig`（交易品种管理页面），此处不再包含 `is_active` / `allow_open` 字段。

## 核心操作

### 交易所筛选

- 搜索栏支持按交易所下拉筛选（静态选项：上期所/大商所/郑商所/中金所/广期所）
- 支持按品种代码关键词搜索

### 统计信息

- 点击"统计信息"按钮弹出 `ContractStatsDialog`
- 显示：合约总数、按交易所分布数量

---

## 数据同步

### 自动同步

- **收盘任务（15:02）**：自动调用 `sync_contract_list_from_tqsdk()`
- 从 TqSdk 拉取全市场主力合约
- 对比数据库，若主力合约变更 → 标记 `is_rollover_needed=True`（通知开盘任务移仓）

### 手动同步

```bash
cd backend
python manage.py sync_contracts          # 从 TqSDK 同步
python manage.py sync_contracts --seed   # 用内置种子数据初始化（无 TqSDK 环境）
```
