# 合约管理功能说明

## 功能概述

合约管理模块用于维护期货交易系统中的交易标的池。系统从 TqSdk 同步全市场主力合约，管理员可在此激活/停用合约、查看合约元数据、控制交易开关。

---

## 核心字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `exchange` | 交易所代码 | SHFE（上期所）、DCE（大商所）、CZCE（郑商所）、CFFEX（中金所）、GFEX（广期所） |
| `product_code` | 品种代码（不带年份） | rb、MA、IF |
| `symbol` | 当前主力合约代码 | SHFE.rb2610 |
| `name` | 合约中文名称 | 螺纹钢2605 |
| `volume_multiple` | 合约乘数（吨/手或点/手） | 螺纹钢=10，股指IF=300 |
| `price_tick` | 最小变动价位 | 螺纹钢=1，黄金=0.02 |
| `min_position` | 交易所限制最小开仓手数 | 通常=1，部分品种可能有特殊限制 |
| `night_trading` | 是否有夜盘交易 | true/false |
| `is_active` | 是否启用交易（核心开关） | true=策略会订阅和交易此品种 |
| `allow_open` | 是否允许开仓 | false=只能平仓不能开新仓 |

---

## 核心操作

### 激活/停用合约

- **行级操作**：每行右侧有"激活/停用"切换按钮
- **批量操作**：支持多选后批量激活/停用（后端 API：`/api/contracts/activate/`、`/api/contracts/deactivate/`）
- **影响**：只有 `is_active=True` 的合约，策略主循环才会订阅和计算指标

### 交易所筛选

- 搜索栏支持按交易所下拉筛选
- 交易所列表动态从后端获取（`GET /api/contracts/exchanges/`）

### 统计信息

- `GET /api/contracts/statistics/`
- 返回：总合约数、激活数、停用数、按交易所分布

## 数据同步

### 自动同步

- **收盘任务（15:02）**：自动调用 `sync_contract_list_from_tqsdk()`
- 从 TqSdk 拉取全市场主力合约
- 对比数据库，若主力合约变更 → 标记 `is_rollover_needed=True`

### 手动同步

```bash
cd backend
python manage.py sync_contracts
```
