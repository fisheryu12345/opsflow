# Dashboard 绩效看板功能说明

## 功能概述

Dashboard 是 FT 量化期货交易系统的首页，集中展示账户绩效、风险指标、持仓状态和可视化图表。数据来源于后端三层绩效体系（L1 日权益快照 / L2 滚动指标 / L3 账户总览）。

## 图表组件

### 1. 品种胜率统计（柱状图）

- **接口**：`GET /api/stock/symbol-win-rate/?account=1`
- **数据**：按 product_code 分组统计平仓记录
- **展示**：
  - X 轴：品种代码
  - Y 轴：胜率（%）
  - 柱色：盈利绿色 / 亏损红色
  - 附加信息：交易次数、多空分布、总盈亏

### 2. 日历热力图

- **接口**：`GET /api/stock/daily-returns-calendar/?account=1`
- **数据**：日权益快照的 `daily_return` 字段
- **展示**：
  - X 轴：日期（1-31日）
  - Y 轴：月份（1-12月）
  - 颜色：深红（>5%盈利）→ 浅红 → 白 → 浅绿 → 深绿（<-5%亏损）
  - Tooltip：悬停显示具体日期和收益率

### 3. 资金曲线

- **接口**：`GET /api/stock/equity-snapshots/?account=1&ordering=trade_date`
- **数据**：每日 `balance` 权益值
- **展示**：折线图，带面积填充

### 4. 资金回撤曲线

- **接口**：`GET /api/stock/drawdown-curve/?account=1`
- **数据**：`peak_equity` 和 `drawdown_pct`
- **展示**：
  - Y 轴固定最大值为 0
  - 面积填充红色渐变
  - MarkLine 标注最大回撤位置
  - MarkPoint 标记创出新高时点

### 5. 平仓盈亏曲线

- **数据**：`ClosedPositionRecord` 按日期聚合的累计盈亏
- **展示**：折线图，盈利段绿色、亏损段红色

### 6. 多窗口绩效雷达图

- **组件**：`MultiWindowRadarChart.vue`
- **接口**：`GET /api/stock/rolling-metrics/?window_days__in=20,60,120,250`
- **维度**：夏普比率 / 索提诺比率 / 波动率 / 胜率 / 盈亏比
- **展示**：4 个窗口（20/60/120/250日）的雷达图叠加对比

## 响应式设计

- **桌面端**：完整标签、较大图表
- **平板/手机**：简化标签、每5天显示一个刻度、较小符号尺寸
