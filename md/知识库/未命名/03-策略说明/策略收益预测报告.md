# 策略收益预测报告（2026-05-09）

> 本报告基于当前策略代码逻辑、历史趋势跟踪 CTA 表现基准、中国商品期货市场特征，
> 对策略在国内流动性好的主力合约上的预期表现进行量化估算。
> 
> **免责声明**：以下所有数据均为基于历史统计和策略参数的估算，不构成未来收益承诺。
> 实际收益受市场环境、执行偏差、交易成本等多因素影响。

## 一、策略特征概览

## 四、加仓对收益的影响分析

### 4.1 金字塔加仓的杠杆效应

| 状态 | 风险暴露 | 占权益比 | 趋势中放大倍数 |
|------|---------|---------|-------------|
| 1 Unit（开仓） | ¥4,000 | 0.4% | 基准 |
| 2 Unit（一次加仓） | ¥8,000 | 0.8% | 2.0× |
| 3 Unit（满仓） | ¥12,000 | 1.2% | 3.0× |

**对年化收益的影响**（估算）：

| 情况 | 不加仓 | 当前加仓 | 差异 |
|------|-------|---------|------|
| 强趋势年 | 12% | 18%~24% | +6%~+12% |
| 弱趋势年 | 4% | 3%~5% | -1%~+1% |
| 震荡年 | -3% | -5%~-8% | -2%~-5%（加剧亏损） |
| 年化（长期） | 8%~12% | 12%~18% | **+3%~+5%** |

> 加仓在趋势中显著放大收益，但在震荡中也会加剧亏损。长期来看，正确加仓（只在趋势中加）能提升年化约 3%~5%。

### 4.2 加仓对胜率的影响

| 交易类型 | 笔数占比 | 胜率 | 单笔平均盈亏 |
|---------|---------|------|------------|
| 1 Unit（仅开仓） | ~55% | 42% | +¥6,000 / -¥4,000 |
| 2 Unit（一次加仓） | ~30% | 38% | +¥16,000 / -¥8,000 |
| 3 Unit（满仓加仓） | ~15% | 35% | +¥30,000 / -¥12,000 |

> 加仓次数越多，胜率越低（因为需要趋势延续才能盈利），但单笔盈利越大。

## 五、不同账户规模的预期收益

### 5.1 收益与规模关系

| 初始权益           | 每 Unit 风险  | 最大并行品种数  | 预期年化        | 最大回撤        | 适合性         |
| -------------- | ---------- | -------- | ----------- | ----------- | ----------- |
| ¥300,000       | ¥4,000     | 3~5      | 10%~18%     | 20%~30%     | 偏小，品种集中度高   |
| ¥500,000       | ¥4,000     | 6~9      | 12%~20%     | 18%~25%     | 可运行核心品种     |
| **¥1,000,000** | **¥4,000** | **8~12** | **12%~20%** | **15%~22%** | **推荐最小规模**  |
| ¥2,000,000     | ¥4,000     | 12~18    | 10%~18%     | 12%~18%     | 分散充分        |
| ¥5,000,000     | ¥4,000     | 18~22    | 8%~15%      | 10%~15%     | 全部品种，边际收益递减 |

> **¥1,000,000 是最小推荐规模**。低于此规模，可交易品种过少，分散化不足，实际波动率会显著高于理论值。

### 5.2 规模收益递减原因

当账户规模增大时：
1. **流动性限制**：大资金在部分品种上滑点增加
2. **分散化上限**：超过 15~18 个品种后，新增品种相关性高，分散效果递减
3. **冲击成本**：部分品种（如苹果、鸡蛋）流动性有限，大单影响成交价

## 九、综合结论

### 9.1 预期收益汇总

| 指标 | 中性估计 | 乐观估计 | 悲观估计 |
|------|---------|---------|---------|
| 年化收益率 | 12%~18% | 20%~30% | 2%~8% |
| 夏普比率 | 0.5~0.7 | 0.8~1.0 | 0.1~0.3 |
| 最大回撤 | 20%~25% | 15%~20% | 28%~35% |
| 胜率 | 38%~42% | 42%~48% | 30%~38% |
| 盈亏比 | 2.2~2.5 | 2.5~3.0 | 1.8~2.2 |
| 年度盈利概率 | 65%~75% | 75%~85% | 50%~60% |
| 月度盈利概率 | 55%~65% | 60%~70% | 45%~55% |

### 9.2 关键结论

1. **趋势跟踪策略长期有效**：在中国商品期货市场，趋势跟踪是最经典的 CTA 策略之一。本策略在参数设定上相对谨慎（MA 过滤 + 保本兜底），长周期夏普比率预计在 **0.5~0.7** 之间。

2. **收益集中在少数交易**：预计约 60% 的利润来自 **5%~10%** 的最佳交易。这是所有趋势跟踪系统的共同特征，不要因连续亏损而放弃策略。

3. **最大回撤不可避免**：预计每 2~3 年出现一次 20%+ 的回撤，持续 4~8 个月。这是趋势跟踪策略的固有特征，不是策略失效。

4. **¥1,000,000 为最小推荐规模**：低于此规模无法充分分散品种，实际波动率会显著高于理论值。

5. **核心风险是震荡磨损**：连续 6~10 笔止损在一年内可能出现 2~3 次。心理承受能力是实盘成功的关键。

### 9.3 实盘建议

| 建议 | 说明 |
|------|------|
| **保证金的 2~3 倍资金** | 1,000,000 权益中约 30%~50% 用于保证金，其余为安全垫 |
| **至少运行 2 年** | 短于 2 年的评估周期无法反映策略的真实表现 |
| **关注最大回撤而非月度收益** | 趋势跟踪的回撤恢复需要时间，不要因回撤而中断策略 |
| **不要频繁优化参数** | 当前参数设定合理，过度优化会导致样本外表现恶化 |
| **建立"震荡期"心理预算** | 预期每年有 3~5 个月的亏损期，属于正常波动 |

## 十、与原版海龟系统的深度对比

### 10.1 海龟系统在中国市场的预期表现

原版海龟（1980s 美国市场）以 15%~25% 年化著称，但那是在趋势更流畅、竞争更少、滑点更低的环境下。如果原封不动搬到今天的中国商品期货市场，预期需要向下修正：

| 调整因素 | 影响 | 调整幅度 |
|---------|------|---------|
| 1980s 美国 → 2020s 中国市场环境差异 | 趋势流畅度下降、假突破增多 | -3%~-5%/年 |
| 无 MA 过滤，突破信号多 ~40% | 震荡期额外磨损 | -2%~-4%/年 |
| 无动态止损，固定 2N | 强趋势中被洗出概率增加 | -1%~-2%/年 |
| 无保本机制 | 利润回吐增加 | -1%~-2%/年 |
| 中国商品波动率高于美国 | 趋势幅度更大 | +1%~+2%/年 |
| **综合调整** | | **-6%~-11%/年** |

**中国市场下的海龟系统预期**：

| 场景 | 原版海龟（美国 1980s） | 海龟（中国当前市场估算） |
|------|-------------------|-------------------|
| 强趋势年 | 30%~40% | 20%~30% |
| 中性年 | 15%~25% | 8%~15% |
| 弱趋势年 | 0%~10% | -5%~5% |
| 极端震荡年 | -10%~0% | -12%~-5% |
| **长期年化** | **~20%** | **~10%~14%** |

### 10.2 逐项对比

| 维度        | 原版海龟                  | 当前系统                           | 差异分析                     |
| --------- | --------------------- | ------------------------------ | ------------------------ |
| **开仓信号**  | 唐奇安 20 日突破（无过滤）       | 唐奇安突破 + MA10/20/40 排列过滤        | 海龟信号多 ~40%，震荡期磨损更大       |
| **胜率**    | ~33%~35%              | ~38%~42%                       | MA 过滤提升约 5~7 个百分点        |
| **止损方式**  | 固定 2N（入场价锚定）          | 动态 2.0~3.0 ATR（跟踪最高/最低价）       | 海龟风险预先确定；当前系统让利润奔跑更充分    |
| **加仓规则**  | 每 0.5N 加 1 单位，最多 4 单位 | 每 0.5~1.0 ATR 加 1~2 单位，最多 3 单位 | 海龟加仓更激进，趋势中收益更高，回调时回撤也更大 |
| **保本机制**  | 无                     | 盈利 > 2.5×ATR 后 tick×2 兜底       | 当前系统极端行情中有本金保护           |
| **趋势过滤**  | 无                     | 连续线性 factor [0, 0.5]           | 当前系统自适应调整止损宽度            |
| **双保险风控** | 收盘后次日开盘处理             | 收盘价止损 + 14:57 临收盘紧急平仓          | 当前系统多一道当日保护              |
| **品种覆盖**  | 美国期货市场 1980s 活跃品种     | 中国商品 22 个主力合约                  | 覆盖范围相当，市场不同              |

### 10.3 不同市场环境下的表现差异（中国市场基准）

| 市场环境 | 海龟系统（中国市场） | 当前系统 | 差异原因 |
|---------|---------------|---------|---------|
| **强趋势年** | 20%~30% | 20%~30% | 基本持平。海龟 4 单位加仓收益更高，但当前动态止损减少被洗出，两者抵消 |
| **中性年** | 8%~15% | 12%~18% | MA 过滤减少假突破亏损，动态止损在震荡中偏紧（2.0 ATR）保护本金 |
| **弱趋势年** | -5%~5% | 2%~8% | MA 过滤跳过大量假信号，显著减少磨损 |
| **极端震荡年** | -12%~-5% | -8%~-2% | 保本兜底 + MA 过滤双重保护，亏损上限更低 |
| **最大回撤** | 28%~38% | 18%~28% | 中性年份亏损更小，回撤深度浅约 30% |
| **年度亏损概率** | 35%~45% | 25%~35% | MA 过滤将亏损年份概率降低约 10 个百分点 |

### 10.4 优化的本质：不是更高收益，而是更高存活率

**核心结论：当前系统的优化目标不是收益率数字，而是策略的实盘可执行性。**

海龟系统在中国市场的核心问题不是"赚得不够多"，而是：

1. **回撤太深（28%~38%）**：大多数个人投资者在 25%+ 回撤时心理崩溃，放弃策略
2. **震荡期亏损太重（-12%~-5%）**：连续 10+ 笔止损后，信心和资金双重耗尽
3. **无利润保护**：一次 ATR 跳升可以让数月的利润在几天内回吐殆尽

当前系统针对这三点做了针对性改进：

| 海龟的痛点 | 当前系统的改进 | 效果 |
|-----------|-------------|------|
| 震荡期频繁假突破 | MA 过滤减少 ~40% 的假信号 | 震荡年少亏 5%~10% |
| 趋势转折时无保护 | 保本兜底锁定已实现利润 | 极端行情减少回撤 5%~10% |
| 所有行情统一止损宽度 | 动态 factor 自适应 | 弱趋势紧止损（2.0 ATR），强趋势宽止损（3.0 ATR） |
| 无临收盘保护 | 14:57 临收盘平仓 | 避免隔夜跳空带来超出预期的亏损 |

**收益代价**：这些改进并非免费。MA 过滤会错过一些真突破，保本兜底可能在震荡洗盘中提前出场，动态止损在强趋势中的 3.0 ATR 意味着更大的浮动亏损容忍度。综合来看，当前系统比海龟系统在**年化收益率上低约 1%~3%/年**，但这是用可控的成本换取**实盘存活率的大幅提升**。

### 10.5 选哪个？

| 如果你的目标是...         | 适合的系统                       |
| ------------------ | --------------------------- |
| 最大化收益，能承受 35%+ 回撤  | 原版海龟                        |
| 稳定的长期复利，回撤可控       | **当前系统**                    |
| 机构资金，风控优先          | 当前系统（保本兜底 + 双保险风控更符合机构合规要求） |
| 个人投资者，心理承受能力一般     | **当前系统**（浅回撤 + 高胜率，更容易坚持执行） |
| 大资金（500 万+），追求绝对收益 | 当前系统（海龟 4 单位加仓在大资金中滑点成本过高）  |

**最终建议**：对于大多数实盘交易者，当前系统的综合性价比更高。收益略低 1%~3%/年，但换来了更浅的回撤、更高的胜率、更低的年度亏损概率。一个能坚持执行 10 年的"中等"策略，远好于一个在大回撤中被放弃的"优秀"策略。

## 附录：计算方法说明

### A. 年化收益率计算

```
年化收益率 = (总盈利 - 总亏损) / 初始权益 × (365 / 回测天数)

其中：
  总盈利   = 胜笔数 × 平均盈利
  总亏损   = 负笔数 × 平均亏损
  胜笔数   = 年交易笔数 × 胜率
  负笔数   = 年交易笔数 × (1 - 胜率)
```

### B. 夏普比率计算

```
夏普比率 = (年化收益率 - 无风险利率) / 年化波动率

无风险利率：2%（参考 10 年期国债）
年化波动率：基于策略月度收益标准差 × √12
```

- 原版海龟法则 1984~1986 年实盘记录（经中国市场环境调整后用于对比分析）
- 国内 CTA 策略 2010~2023 年行业统计数据
- 各大宗商品板块 2015~2025 年历史波动率数据
- 南华商品指数历史走势
- 中国商品期货市场与 1980s 美国期货市场的结构性差异分析（用于海龟系统的中国市场预期修正）
