# OpsFlow 模型表关系说明

> 严格基于 `backend/opsflow/models/` 目录下的全部 Django Model 源代码生成

## 一、总览

OpsFlow 共有 **15 个数据模型**，分布在 8 个模型文件中，使用 `ops_` 前缀的表名。

| 模型文件 | 模型 | 物理表 | 说明 |
|---------|------|--------|------|
| template.py | FlowTemplate | `ops_flow_template` | 编排模板主表 |
| template.py | TemplateVersion | `ops_template_version` | 模板版本历史 |
| template.py | TemplateNode | `ops_template_node` | 模板节点（反范式化） |
| template.py | TemplateCollect | `ops_template_collect` | 用户模板收藏 |
| template.py | TemplateCategory | `ops_template_category` | 模板分类 |
| execution.py | FlowExecution | `ops_flow_execution` | 执行实例 |
| execution.py | ExecutionNode | `ops_execution_node` | 执行节点 |
| execution.py | ExecutionScheme | `ops_execution_scheme` | 执行方案 |
| execution.py | AutoRetryStrategy | `ops_auto_retry_strategy` | 自动重试策略 |
| execution.py | NodeTimeoutConfig | `ops_node_timeout_config` | 节点超时配置 |
| execution.py | NodeExecutionTrace | `ops_node_trace` | 节点执行轨迹 |
| schedule.py | SchedulePlan | `opsflow_schedule_plan` | 调度计划 |
| project.py | OpsProject | `ops_project` | 项目（数据隔离） |
| project.py | ProjectMember | `ops_project_member` | 项目成员 |

| audit.py | OpsLog | `ops_log` | 执行审计日志 |

| webhook.py | WebhookConfig | `ops_webhook_config` | Webhook 回调配置 |
| webhook.py | WebhookLog | `ops_webhook_log` | Webhook 投递日志 |

## 二、核心模型字段详解

### FlowTemplate — `ops_flow_template`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | AutoField | PK | |
| name | CharField(200) | NOT NULL | 模板名称 |
| pipeline_tree | JSONField | default=dict | Pipeline 定义 |
| target_hosts | JSONField | default=list | 目标主机 |
| global_vars | JSONField | default=dict | 全局变量 |
| is_draft | BooleanField | default=True | 是否为草稿 |
| ai_original_tree | JSONField | default=dict | AI 原始树 |
| category | CharField(64) | blank=True | 分类 |
| tags | JSONField | default=list | 标签 |
| description | CharField(500) | blank=True | 描述 |
| hook_variables | JSONField | default=dict | 变量提升配置 |
| project | FK→OpsProject | CASCADE | 所属项目 |
| created_by | FK→User | PROTECT | 创建人 |
| version | IntegerField | default=1 | 当前版本 |
| snapshot | JSONField | nullable | 发布快照 |
| is_public | BooleanField | default=False | 是否公开 |
| project_scope | JSONField | default=list | 可见项目范围 |

### FlowExecution — `ops_flow_execution`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | AutoField | PK | |
| template | FK→FlowTemplate | PROTECT | 关联模板 |
| project | FK→OpsProject | CASCADE | 所属项目 |
| status | CharField(16) | choices | pending/running/paused/completed/failed/cancelled |
| node_status | JSONField | default=dict | 节点状态映射 |
| state_tree | JSONField | nullable | 状态树快照 |
| context | JSONField | default=dict | 运行时上下文 |
| template_snapshot | JSONField | nullable | 创建时冻结的快照 |
| current_node | CharField(200) | blank=True | 当前执行节点 |
| schedule_plan | FK→SchedulePlan | SET_NULL | 关联调度 |
| parent_execution | FK→self | SET_NULL | 父执行（子流程） |
| is_subprocess | BooleanField | default=False | 独立子流程 |
| excluded_nodes | JSONField | default=list | 排除的节点 ID |
| started_at/ended_at | DateTimeField | nullable | 起止时间 |
| created_by | FK→User | PROTECT | 执行人 |

### NodeExecutionTrace — `ops_node_trace`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| execution | FK→FlowExecution | CASCADE | 所属执行 |
| node_id | CharField(200) | | 节点 ID |
| status | CharField(16) | default=pending | 终态 |
| status_history | JSONField | default=list | 状态变更历史 |
| entered_at/exited_at | DateTimeField | nullable | 起止时间 |
| duration_ms | IntegerField | nullable | 耗时 |
| inputs/outputs | JSONField | default=dict | 输入/输出 |
| error | TextField | blank=True | 错误信息 |
| retry_count/max_retries | IntegerField | default=0 | 重试计数 |
| log_file_path | CharField(500) | blank=True | 日志文件路径 |

**约束：** unique_together = [(execution, node_id, retry_count)]

## 三、模型关系图

```
OpsProject ──┬── ProjectMember                (1:N 成员)
              ├── ProjectEnvironmentVariable   (1:N 环境变量)
              ├── FlowTemplate                (1:N 模板)
              ├── FlowExecution               (1:N 执行)
              └── SchedulePlan                (1:N 调度)

FlowTemplate ─┬── TemplateVersion              (1:N 版本历史)
               ├── TemplateNode               (1:N 模板节点)
               ├── TemplateCollect             (1:N 收藏)
               ├── FlowExecution              (1:N 执行)
               ├── SchedulePlan               (1:N 调度)
               ├── ExecutionScheme             (1:N 执行方案)
               ├── WebhookConfig              (1:N Webhook)
               └── TemplateCategory           (N:1 分类)

FlowExecution ─┬── ExecutionNode              (1:N 执行节点)
                ├── NodeExecutionTrace        (1:N 节点轨迹)
                ├── AutoRetryStrategy         (1:N 重试策略)
                ├── NodeTimeoutConfig         (1:N 超时配置)
                ├── OpsLog                    (1:N 审计日志)
                ├── WebhookLog                (1:N Webhook日志)
                ├── SchedulePlan              (N:1 调度)
                ├── FlowExecution (self)      (N:1 父执行)
                └── FlowTemplate              (N:1 模板)
```
